# Screenshot Naming Convention

Use a fast, consistent pattern:

## Pattern
<SECTION>-<STEP>_<short-description>.png

## Examples
- 01-01_hyperv-new-vm.png
- 02-03_windows-setup-disk.png
- 03-04_ipconfig-dc1.png
- 04-02_adds-install-wizard.png

## Notes
- SECTION maps to your repo folders (01-vm-create, 02-install, etc.)
- STEP increments within that section
- Keep descriptions short and searchable
